# Program Pemesanan Makanan Menggunakan Bahasa C++

## Deskripsi
Project ini merupakan program **pemesanan makanan sederhana berbasis C++** yang berjalan di terminal/console. Program ini dirancang untuk menangani input data pengguna, pemilihan menu makanan, penentuan jumlah porsi, hingga perhitungan total pembayaran secara otomatis.

Project ini sangat cocok sebagai referensi latihan dasar pemrograman C++ bagi pemula yang ingin memahami konsep:
- Input & Output (`cin`, `cout`)
- Percabangan (`switch-case` atau `if-else`)
- Perulangan (`do-while` untuk pengulangan pesanan)
- Operasi aritmatika sederhana untuk kalkulasi harga

---

## Fitur Utama
* **Input Data Pengguna:** Mencatat identitas pemesan (Nama dan Prodi).
* **Pilihan Menu Makanan:**
    * Bakso
    * Mie Ayam
    * Otak-Otak
* **Validasi Input:** Sistem akan mengecek apakah menu yang dipilih tersedia.
* **Manajemen Porsi:** Pengguna dapat menentukan jumlah porsi untuk setiap menu.
* **Kalkulasi Otomatis:** Menghitung total harga secara instan.
* **Nota Pembayaran:** Menampilkan ringkasan pesanan di akhir program.

---

## Teknologi yang Digunakan
* **Bahasa Pemrograman:** C++
* **Compiler:** g++ (MinGW / GCC)
* **Standar Library:** `<iostream>`, `<string>`
