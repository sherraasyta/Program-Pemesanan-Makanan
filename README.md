# Program Pemesanan Makanan Menggunakan Bahasa C++

## Deskripsi
Project ini merupakan program **pemesanan makanan sederhana berbasis C++** yang berjalan di terminal/console.  
Program digunakan untuk melakukan input data pengguna, memilih menu makanan, menentukan jumlah porsi, dan menghitung total pembayaran secara otomatis.

Project ini cocok sebagai latihan dasar pemrograman C++ dengan konsep:
- Input & output
- Percabangan (switch)
- Perulangan (do-while)
- Operasi aritmatika sederhana

---

## Fitur Utama
- Input data pengguna (nama dan prodi)
- Pilihan menu makanan
  - Bakso
  - Mie Ayam
  - Otak-Otak
- Validasi pilihan menu
- Input jumlah porsi
- Perhitungan total harga
- Menampilkan nota pembayaran

---

## Teknologi yang Digunakan
- **Bahasa Pemrograman**: C++
- **Compiler**: g++ (GNU Compiler Collection)
- **Lingkungan**: Terminal / Command Prompt
